Description

In this stage, you will enhance your CLI-based assistant by enabling it to accept user input from the console, send it to the GPT API, and display the assistant's response. Additionally, you will calculate and display the total token cost associated with each interaction.

Your first task in this stage is to prompt the user for input using the console. Display a message saying "Enter a message:" to guide the user. Capture their input and prepare it to be sent to the GPT API.

Next, send the user's input as a message to the GPT API. This will prompt the assistant to generate a response based on the user's input.

Once the assistant's response is received, display it on the console. Precede the response with "Assistant:" to clearly distinguish between the user's input and the assistant's response. Also, output the user's message in this format: "You:" before assistant's response.

As an added feature, calculate the total token cost of each interaction. This includes both the input (user's message) and output (assistant's response) costs. Display this cost on the console following each interaction, formatted as "Cost: $...".

Refer to the example for the output format.

Objectives

In this stage, you should:

  1. Prompt the user for input with "Enter a message:" and capture their input.

  2. Send the user's input to the GPT API.

  3. Display the assistant's response on the console, preceded by "Assistant:".

  4. Calculate and display the total token cost of each interaction, formatted as "Cost: $...".

Examples

The greater-than symbol followed by a space (> ) represents the user input. Note that it's not part of the input.

Example 1:

Enter a message: > What is 5 + 10?
You: What is 5 + 10?
Assistant: 5 + 10 equals 15.
Cost: $0.00003000

Example 2:

Enter a message: > What is the largest ocean?
You: What is the largest ocean?
Assistant: The largest ocean is the Pacific Ocean.
Cost: $0.00002900
